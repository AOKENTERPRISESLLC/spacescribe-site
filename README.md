# spacescribe-site

Static placeholder site for [spacescribe.sol](https://spacescribe.sol.site/).

Single-file HTML + assets, served by GitHub Pages, reached via Bonfida SNS
pointing `spacescribe.sol` at this repo's Pages URL.

## Structure

```
index.html                        single-file page (inline CSS)
assets/scribe-collage.png         full-res circular collage (1858×1858)
assets/scribe-collage-1200.png    web-optimized (1200×1200)
```

No build step. No framework. Edit `index.html` and push.
